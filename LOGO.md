# Woclaw LOGO Design - 启明星

## 品牌故事

**Woclaw** 谐音"启明星"，北极星，指引方向的星光。设计围绕"星空中的明星"展开。

## 设计元素

### 核心意象
- 🌟 **星星** - 夜空中的明亮星体，象征智慧和指引
- ✨ **光芒** - 向四周放射的光芒，象征照亮黑暗
- 🔵 **深蓝背景** - 太空的深邃感
- 🌌 **星空** - 点点星光装饰

### 配色方案
```
主色：
  - #0A1628 (深空蓝 - 背景)
  - #1E3A5F (星空蓝 - 次级)
  - #FFD700 (明星金 - 主色)
  - #FFF8DC (柔光黄 - 发光效果)

辅助色：
  - #4A90D9 (天际蓝)
  - #E8E8E8 (星光白)
```

### 图形设计

```
         *
        ***
       *****
      *******
    ✨ ********* ✨
      *******
       *****
        ***
         *
```

简化版 ASCII LOGO：
```
        ✨
    ✨  ⭐  ✨
      ╔═══╗
    ✨ ║ W ║ ✨    Woclaw - 启明星
      ╚═══╝
    ✨       ✨
```

## 图形规范

### 主要图形
1. **星星主体**：六角或八角星形
2. **光芒**：向四周放射的短线条
3. **字母 W**：居中或偏上，体现品牌名

### 变体设计

**简洁版（用于小图标）**
```
  ✨
```

**完整版（用于封面）**
```
    ·  ✨  ·
  ✨  ⭐  ✨
    ╔═══╗
  ✨ ║ W ║ ✨
    ╚═══╝
  · ✨   ✨ ·
```

- 星星图案居中

- 金色星星

- 使用星星 emoji: ✨ ⭐ 🌟

建议使用 SVG 或 PNG 实现，尺寸：
- 头像：128x128 / 256x256
