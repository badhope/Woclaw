## ✨ Woclawとは？

Woclaw（「明けの明星」と発音）は、**軽量な自律型コンピュータ制御AIアシスタント**です。自然言語の指示を理解し、电脑操控等各种任務を達成するために помощьします。

**核心理念**：AIに人間のようにコンピュータを操作させ、あなたを程序员にさせない。

## 🌟 主な機能

- 🧠 **スマートな意思決定者** - あなたの意図を理解し、执行プランを作成
- 🔧 **完全な制御** - ファイル、プロセス、GUIウィンドウ、ネットワークリクエスト
- 📚 **自己学習** - 間違いから学び、時間が経つにつれて賢くなる
- 🔒 **安全で制御可能** - リスクレベル設定、危険な操作は確認が必要
- 🛠️ **豊富なツール** - 5つの組み込みWorker
- 💬 **マルチプラットフォーム** - CLI、微信、钉钉、飞书...

## 🚀 クイックスタート

### ワンクリックインストール（Windows）

```powershell
# インストールスクリプトをダウンロードして実行
iwr https://raw.githubusercontent.com/badhope/Woclaw/main/scripts/install.ps1 -UseBasicParsing | iex
```

### 手動インストール

```bash
# プロジェクトをクローン
git clone https://github.com/badhope/Woclaw.git
cd Woclaw

# 依存関係をインストール
pip install -e .

# 起動
woclaw chat
```

## 🏗️ アーキテクチャ

```
ユーザーメッセージ
    ↓
┌─────────────────────────────┐
│   Supervisor（意思決定者）     │
│   • 意図を理解               │
│   • タスクを計画             │
│   • 計画を承認               │
│   • 結果を検証               │
└─────────────────────────────┘
    ↓
┌─────────────────────────────┐
│   Workers（作業者）           │
│   • FileWorker - ファイル操作 │
│   • GuiWorker - GUI制御     │
│   • ShellWorker - コマンド   │
│   • WebWorker - HTTP        │
│   • SysWorker - システム    │
└─────────────────────────────┘
```

## 📖 使用例

### ファイル管理

```
あなた：ダウンロードフォルダの画像を日付別に整理して
スター：わかりました！整理を手伝います~

✅ ダウンロードフォルダをスキャン（32枚の画像を発見）
✅ 日付フォルダを作成（2026-03、2026-02...）
✅ ファイルをフォルダに移動
✨ 完了！32枚の画像を整理しました
```

### GUI制御

```
あなた：メモ帳を開いて「こんにちはスター」と入力して保存して
スター：承知しました！今すぐ操作します~

✅ メモ帳を開く
✅ ウィンドウをアクティブ化
✅ テキストを入力
✅ ファイルメニューをクリック
✅ 名前を付けて保存をクリック
✅ ファイルを保存
✨ 完了！ファイルを保存しました
```
